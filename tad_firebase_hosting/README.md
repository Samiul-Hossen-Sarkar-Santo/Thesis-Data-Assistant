# Thesis Data Assistant


A privacy-first, client-side web tool for academic researchers to view, process, and convert data files, and extract text from images using OCR.

## Hosted Site

Access the live application here: [https://thesis-data-assistant.web.app](https://thesis-data-assistant.web.app)

## Features

- **Data Viewer**: Upload and view CSV, Excel (.xlsx/.xls), and JSON files. Search, paginate, and export data in multiple formats.
- **Image Lab**: Upload images (JPG, PNG, WEBP), extract text using OCR (English/Bengali), view image metadata, and convert image formats.
- **Feedback**: Send feature requests or bug reports via email (no data stored).
- **Legal & Disclaimer**: Transparent privacy policy and open-source credits.
- **Support**: Donate via bKash/Nagad to support development.

## Technology Stack

- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) — UI framework
- [PapaParse](https://github.com/mholt/PapaParse) — CSV parsing
- [SheetJS](https://github.com/SheetJS/sheetjs) — Excel/JSON parsing
- [Tesseract.js](https://github.com/naptha/tesseract.js) — OCR engine
- [Exif.js](https://github.com/exif-js/exif-js) — Image metadata
- [Google Fonts](https://fonts.google.com) — Multilingual font support

## Privacy & Data Policy

- All processing is done locally in your browser. No data is uploaded, stored, or tracked.
- Uploaded files and images are erased from memory when you refresh or close the tab.

## How to Use

1. **Clone or download the repository.**
2. Open `index.html` in your browser.
3. Use the navigation tabs to access Data Viewer, Image Lab, Feedback, Legal, and Credits.
4. For support, use the bKash/Nagad numbers in the Credits section.
